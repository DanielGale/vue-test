## Using Vue-Loader

This example shows a vue app using the vue-loader


Run:
 npm install -g @vue/cli-init
 vue init webpack-simple my-app

 Accept Defaults

? Project name my-app
? Project description A Vue.js project
? Author
? License MIT
? Use sass? No

   vue-cli · Generated "my-app".

   To get started:

     cd my-app
     npm install
     npm run dev
